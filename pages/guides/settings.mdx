# Settings

## Auto archive Slack Channels
If 'Auto Channel Archiver' is on, every pull request that your team will close or merge
will automatically archive the corresponding channel in Slack.

## Score rating
You can decide how to compute the score of your engineers in the Axolo leaderboard.

# Notifications
## Github action notifications
Select if you want to receive Github action notifications for your main branch. If selected, action notifications for main/master will be sent in your _axolo channel. 

## Deployment notifications
Select if you want to receive deployment notifications. If selected, deployment notifications will be sent in your _axolo channel. 

## Merged pull request notifications
If 'Merged pull request notification' is on, every merged pull request will automatically
send a notification in your  _axolo channel.
