# Axolo

**Axolo**, [axolo.co](https://axolo.co) makes a temporary slack channel for every github pull requests, it will invites reviewers and assignees to the channel, notify them of relevant events like code comments, comments, Github Actions, Deployments and finally it will archive the channel once you close or merge the pull requests.
